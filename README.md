# GB/T 7714-2015 参考文献著录专家

一个基于中国国家标准 GB/T 7714—2015《信息与文献 参考文献著录规则》的学术编辑专家技能。

## 功能特性

- 严格遵循 GB/T 7714-2015 国家标准
- 支持 14 种文献类型的格式转换
- 自动处理作者姓名格式（中文、西文、拼音）
- 确保标点符号、空格、字体样式完全符合标准
- 提供详细的质量控制自检清单

## 支持的文献类型

1. 专著类 (Monograph) [M]
2. 论文集、会议录 [C]
3. 报告 [R]
4. 学位论文 [D]
5. 标准文献 [S]
6. 档案 [A]
7. 舆图/数据集 [CM]/[DS]
8. 专著中的析出文献
9. 连续出版物 (整本期刊或整份报纸) [J, N]
10. 期刊中析出的文献 [J]
11. 报纸中析出的文献 [N]
12. 专利文献 [P]
13. 电子资源 [EB/OL, DB/OL 等]
14. 预印本文章 [EB/OL]

## 使用方法

提供文献信息（如书名、文章、专利等），系统将自动转换为符合 GB/T 7714-2015 标准的参考文献格式。

### 示例输入
- 普通图书：陈登原. 国史旧闻: 第1卷. 北京: 中华书局, 2000: 29.
- 期刊文章：李炳穆. 韩国图书馆法. 图书情报工作, 2008, 52(6): 6-21.
- 专利文献：邓一刚. 全智能节电器: 200610171314.3. 2006-12-13.

### 输出格式
严格按照 GB/T 7714-2015 标准格式输出，包括正确的标点符号、空格和字段顺序。

## 核心规则

### 标点符号
- 所有著录符号均为**半角符号**
- 符号后通常需跟随一个空格（除圆点`.`后紧接页码等特殊情况外）

### 作者处理
- 3人或以下：全部列出
- 3人以上：列出前3人，后加 `, 等` (中文) 或 `, et al.` (西文)
- 中文：姓在前名在后，不缩写（如：陈登原）
- 西文：姓在前，名在后。**姓全部大写**，名缩写为首字母，不加缩写点（如：EINSTEIN A）

### 文献类型标识
必须使用方括号 `[]` 包裹，如 `[M]`, `[J]`, `[EB/OL]` 等。
